    {{% timeline-item date="2017-11-29" month="November" title="SeedStack 17.11 \"Nymphaea\"" %}} SeedStack 17.11 "Nymphaea" will include major improvements throughout the stack with a completely revamped business framework: * Simplification of DDD building-blocks, * Implementation of Domain specifications, * Powerful collection-like repositories, * Automatic translation of specifications into persistence queries, * Support of Java streams from the repository to the interface objects, * Simpler API for paginating results, * Support for key-based pagination, * Automatic import/export of domain data. A new Java Common-Client-Interface (CCI) add-on will also be available to interact with legacy systems (like mainframe transactions). {{% /timeline-item %}} {{% timeline-item date="2017-07-27"
