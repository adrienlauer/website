## Classic Web application

This sample demonstrates a full end-to-end Web application:

* Simple domain code,
* JPA persistence,
* REST resources,
* Assemblers,
* Pagination,
* Static resources serving,
* W20 UI.

GitHub: https://github.com/seedstack/samples/tree/master/full-apps/store-webapp
